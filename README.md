# 🛠️ Estruturas de Dados e Algoritmos em C

Este repositório contém as implementações desenvolvidas durante o curso de **Análise e Desenvolvimento de Sistemas (FATEC)**. O foco é o estudo de alocação de memória, recursividade e análise de eficiência algorítmica.

---

## 📂 Conteúdo do Repositório

### 🏗️ Estruturas de Dados
* **[🔄 Fila Circular](./Fila-Circular):** FIFO com aproveitamento de memória via aritmética modular.
* **[📚 Pilha (Stack)](./Pilha):** Lógica LIFO utilizando vetor estático.
* **[🔗 Lista Ligada](./Lista-Ligada):** Gerenciamento dinâmico de nós com `malloc` e ponteiros.

### ⚡ Algoritmos de Ordenação
* **[🫧 Bubble Sort](./Ordenacao-BubbleSort):** Ordenação simples por trocas adjacentes.
* **[📥 Insertion Sort](./Ordenacao-InsertionSort):** Ordenação por inserção, eficiente para pequenos conjuntos.
* **[🎯 Selection Sort](./Ordenacao-SelectionSort):** Ordenação por seleção do menor elemento.
* **[⚡ Merge Sort](./Ordenacao-MergeSort):** Algoritmo avançado de alta performance (Dividir e Conquistar).

---

## 🧠 Conceitos Aplicados
- **Alocação Dinâmica:** Manipulação de memória em tempo de execução.
- **Recursividade:** Implementação de funções que chamam a si mesmas (Merge Sort).
- **Análise de Performance:** Monitoramento de trocas e comparações para medir eficiência.
